## Как создаются аккаунты

**Не через самостоятельную регистрацию.** Только **админ** (Руслан) создаёт пользователя через админку:

1. Вводит ФИО + роль (teacher/manager/curator/director)
2. Генерирует временный пароль (показывается один раз, сохраняется в argon2id)
3. Отдаёт учителю этот пароль (лично или в защищённом канале)
4. Учитель при первом входе меняет пароль на свой

При импорте 31 активного учителя:
- Создаются записи в `teachers` (UUID из журнала)
- **Не создаются автоматически** записи в `users` — это делает админ при первой выдаче доступа, чтобы не плодить "мёртвых" учёток
