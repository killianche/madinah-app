## 🗓 Милстоуны

### v1 (задеплоен)

- История по учителям в карточке (v_student_teacher_stats)
- Смена учителя manager/curator/admin с audit_log

- **Расширенные статусы** ученика (paused/graduated/dropped) + смена с причиной
- **Серая зона** (stale/skipping) через `v_student_attention`

- **История смен статуса** на карточке ученика
- **Краткая аналитика** сверху карточки (всего/проведено/штраф/отмены/период)

- **Удаление «вне графика»** (scheduled_date/scheduled_time), миграция 0005
- **Роль `head`** (руководитель учителей) + 3 аккаунта
- **Бейдж серой зоны** на карточке — сигнал учителю сменить статус

### v4 (задеплоен)
- Tailwind палитра из tokens.jsx (moss/crimson/stone/charcoal…)
