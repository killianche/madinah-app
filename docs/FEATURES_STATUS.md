## 🗓 Милстоуны

### v1 (задеплоен)
Инфра, auth, базовые экраны учителя/менеджера, импорт CSV, admin, 38 аккаунтов.

### v2 (задеплоен)
История по учителям в карточке, смена учителя manager/curator/admin, view `v_student_teacher_stats`.

### v3 (в работе)
- Расширенные статусы ученика (paused/graduated/dropped/archived)
- Серая зона (stale/skipping) + страница куратора «Требует внимания»
- История смен статуса на карточке
- Краткая аналитика сверху карточки
- Удаление «вне графика» (scheduled_date/scheduled_time)
- Роль `head` + 3 аккаунта

### v4 (следующая)
- Правка/удаление уроков задним числом (manager/curator)
- Редактирование профиля ученика
- Нормализация +7/8 при логине
- Смена пароля при первом входе
