# Madinah App — инфраструктура (чек-лист покупки)

Хостинг: **TimeWeb Cloud** (timeweb.cloud — не путать с timeweb.com/ru — это разные продукты).
Датацентр: **Москва** (обязательно для 152-ФЗ).

---

## 1. Домен

Используем **существующий `madinah.ru`** (зарегистрирован на reg.ru), поддомен:

- [ ] **`app.madinah.ru`** — точка входа в приложение
- Новый домен покупать **НЕ нужно**
- DNS настраивается в панели reg.ru: добавить **A-запись** `app → IP VPS` (IP дам после покупки хостинга)
- SSL: Let's Encrypt через Caddy/certbot — бесплатно, авто-продление

---

## 2. VPS (Next.js приложение)

**Тариф:** TimeWeb Cloud VDS → **"Разработчик" или "Старт+"**

| Параметр | Минимум | Рекомендую |
|---|---|---|
| CPU | 2 vCPU | 2 vCPU |
| RAM | 2 GB | 4 GB |
| SSD | 40 GB NVMe | 60 GB NVMe |
| ОС | Ubuntu 24.04 LTS | Ubuntu 24.04 LTS |
| Локация | Москва | Москва |
| Цена/мес | ~400₽ | ~700₽ |

Под 32 учителя и 2 000 учеников с запасом хватит 2/4/60.

---

## 3. Managed PostgreSQL

**Тариф:** TimeWeb Cloud → Базы данных → PostgreSQL 16 → **"Начальный"**

| Параметр | Значение |
|---|---|
| Версия | PostgreSQL 16 |
| CPU/RAM | 1 vCPU / 2 GB |
| Диск | 20 GB SSD (автоувеличение) |
| Бэкапы | ежедневные, хранение 7 дней (включено) |
| Цена/мес | ~600₽ |

**Почему managed, а не свой на VPS:**
- Бэкапы автоматом
- Мониторинг
- Обновления версий без боли
- При падении VPS база цела

---

## 4. Объектное хранилище (S3-совместимое)

Для бэкапов БД (дубликат поверх managed-бэкапов) и аватарок/вложений в будущем.

- TimeWeb Cloud S3 — ~100₽/мес за 50 GB
- Можно отложить до V2, не критично на старте

---

## 5. SSL

- **Let's Encrypt** — бесплатно, авто-продление через certbot/Caddy
- На TimeWeb если брать их "Хостинг сайтов" — SSL в комплекте, но нам это не нужно (мы на VPS)

---

## 6. Мониторинг и логи

- **UptimeRobot** (бесплатно) — пинг главной страницы каждые 5 минут
- **Sentry** (бесплатный план) — отлов ошибок фронта и бэка
- **Grafana + Prometheus** — позже, когда взлетит

## Порядок покупки (что делать тебе)

1. Зарегистрироваться на **timeweb.cloud** (если ещё нет)
2. Купить VPS — Ubuntu 24.04, Москва, тариф 2/4/60
3. Купить Managed PostgreSQL — PG16, Москва, начальный
4. Прислать мне:
   - IP VPS + SSH-ключ (либо root-пароль, я поставлю свой ключ)
   - URL + логин/пароль managed Postgres
   - Доступ в reg.ru к DNS-панели `madinah.ru` — либо добавить A-запись по моей инструкции сам:
     `Тип: A, Имя: app, Значение: <IP VPS>, TTL: 3600`

После этого в первый день настрою сервер, накачу миграции, запущу импорт.
